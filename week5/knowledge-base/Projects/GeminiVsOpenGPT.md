# GeminiVsOpenGPT

A web application to compare responses between Google Gemini 1.5 Pro and OpenAI GPT-4o side-by-side in real-time. Built using Gradio and FastAPI, this tool helps users explore how both models respond to the same prompt with custom decoding parameters.

## 🚀 Features

* Real-time chatbot interface for both Gemini and GPT-4o.

* Adjustable parameters for GPT inference (temperature, top-p, max tokens, etc.).

* Supports web search toggle for GPT.

* Async response streaming.

* REST endpoint for health check.
